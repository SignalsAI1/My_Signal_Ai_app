🚀 Про Проект

My Ai Signal - це повноцінна веб-платформа для AI-сигналів Forex трейдингу з інтеграцією Telegram Mini App та брокерським стилем аналізу.

Система використовує мульти-сурсний двигун даних з автоматичним переключенням між джерелами для максимальної стабільності.

🏦 Брокерські Сигнали

EMA 50/200, RSI, MACD, Bollinger Bands з професійною точністю

🔄 Мульти-Сурс Дані

Pocket API → Twelve Data → WebSocket → Cache з автоматичним відновленням

🎨 Dark Neon UI

Сучасний інтерфейс з glassmorphism ефектами та анімаціями

📱 Telegram Mini App

Мобільна версія для трейдингу на ходу

🔐 Система Верифікації

Безпечна система реєстрації з affiliate інтеграцією

🌐 WebSocket Real-time

Живі дані кожні 2 секунди з індикаторами джерела

🛠️ Як Запустити

1. Клонуйте репозиторій:

git clone https://github.com/SignalsAI1/My_Signal_Ai_app cd My_Signal_Ai_app

2. Встановіть залежності:

npm install

3. Налаштуйте змінні середовища:

cp .env.example .env # Редагуйте .env з вашими API ключами

4. Запустіть сервер:

node backend/index.js

5. Відкрийте: http://localhost:3001
